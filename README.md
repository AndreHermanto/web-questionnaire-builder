## Folder structure

```
├── README.md
├── node_modules
├── package.json
├── .gitignore
├── public
│   └── favicon.ico
│   └── index.html
│   └── manifest.json
└── src
    └── containers
    └── components
    └── routes
    └── actions
    └── reducers
    └── constants
    └── App.js
    └── App.test.js
    └── index.css
    └── index.js
    └── logo.svg
    └── registerServiceWorker.js
```

### Installing

```
yarn install or npm install
```

### Break down into end to end tests

```
npm run test
```

## Deployment
